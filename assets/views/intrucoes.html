Como usar o DataAves

1

Prepare sua planilha Excel seguindo os padrões GEAVES

Organize os dados no formato padrão descrito abaixo. A primeira linha deve conter os cabeçalhos (siglas das variáveis). Cada linha subsequente representa uma repetição. Variáveis de categorias diferentes devem estar em abas separadas do mesmo arquivo.

2

Acesse a aba "Dados" e faça o upload

Clique em "Dados" no menu superior. Arraste o arquivo .xlsx ou clique para selecioná-lo. Escolha qual aba (sheet) deseja verificar — o sistema lista automaticamente todas as abas do arquivo.

3

Confira a estatística descritiva

Após a validação bem-sucedida, acesse a aba "Estatística" para visualizar a distribuição e resumo de cada variável por tratamento e fase experimental.

4

Explore os gráficos

Acesse a aba "Gráficos" para visualizar seus dados em gráficos de linhas, barras e boxplot. O layout é gerado automaticamente com base no número de variáveis detectadas.

5

Gere Relatórios e Exporte

Após explorar visualmente seus dados, utilize os comandos na barra superior. Baixe a estatística limpa clicando em ⬇ Excel ou crie um dossiê completo clicando em 📄 PDF para exportar gráficos e matrizes prontos para publicação.

Padrões obrigatórios

Regras de organização de dados — GEAVES

Estas regras foram estabelecidas para garantir padronização das tabelas. O DataAves verifica automaticamente cada uma delas ao carregar sua planilha.

📊

Use apenas médias das repetições por fase

Coloque a média de cada repetição para cada fase experimental, não os dados brutos individuais. Os dados brutos devem estar em arquivo separado e ser enviados ao professor responsável.

✔ Correto TR=1 · REP=1 · Fase1: BWG=44 · FC=1.477
🏷️

Siglas em inglês, sem símbolos

Todos os cabeçalhos devem ser siglas em inglês. Nunca use símbolos como %, $, *, ° nos títulos das colunas.

✘ Albumen%  |  ✔ alb_p
🔡

Sem espaços, acentos ou palavras completas

Use sempre siglas curtas sem acento e sem espaço. Underline _ é permitido para separar partes da sigla.

✘ Percentagem de álbum  |  ✔ alb_p
🚫

Sem siglas repetidas

Cada variável deve ter uma sigla única no arquivo. Siglas duplicadas causam ambiguidade na análise.

✘ BWG, BWG  |  ✔ bwg1, bwg2
🔲

Use NA para parcelas perdidas

Quando uma observação não puder ser registrada, escreva NA. Nunca deixe a célula em branco.

✘ célula vazia  |  ✔ NA
📑

Variáveis por categoria em abas separadas

Separe variáveis por categoria em abas (sheets) diferentes no mesmo .xlsx. Nome da aba: curto, minúsculas, sem acento.

✘ "Desempenho Produtivo"  |  ✔ desempenho · qualidade · bioquimico

Estrutura da planilha

Colunas obrigatórias e variáveis de resposta

As três primeiras colunas são identificadoras obrigatórias; as demais são as variáveis de resposta mensuradas em cada unidade experimental.

ColunaObrigatoriedadeTipoDescrição / Exemplo
TR Obrigatória Categórico Tratamento — número ou código do grupo experimental (1, 2, 3…)
REP Obrigatória Numérico Repetição — número da baia, gaiola ou boxe (1, 2, 3…)
PER Opcional Numérico Período/fase experimental. Obrigatória quando há mais de uma fase.
VAR_1 … VAR_N Obrigatória Numérico Variáveis de resposta em siglas inglesas. Ex.: BWG, FI, FC, EW, EO, alb_p…
